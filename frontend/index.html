API Reference

All endpoints return JSON. Base URL: http://romemu.local

ROM Image

GET/api/slotsCurrent image status
POST/api/slots/0/upload?chip={type}&label={name}Upload ROM (raw binary body)
GET/api/slots/0/downloadDownload current image
POST/api/slots/0/insertStart emulation
POST/api/slots/0/ejectStop emulation
DELETE/api/slots/0Delete image

System

GET/api/statusSystem status
GET/api/chipSupported chip database
GET/api/statsR/W/erase counters
POST/api/stats/resetReset statistics

GPIO Control

GET/api/gpioPin state
POST/api/gpio/reset/pulsePulse reset {"duration":100}
POST/api/gpio/power/cyclePower cycle {"duration":500,"settle":200}

Example: curl

curl -X POST "http://romemu.local/api/slots/0/upload?chip=4&label=fw" \
     -H "Content-Type: application/octet-stream" --data-binary @firmware.bin
curl -X POST http://romemu.local/api/slots/0/insert
curl -X POST http://romemu.local/api/gpio/reset/pulse -d '{"duration":100}'
curl -N http://romemu.local/api/events
