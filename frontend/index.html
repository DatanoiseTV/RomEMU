API Reference

All endpoints return JSON. Base URL: http://romemu.local

System

GET/api/statusSystem status (uptime, heap, PSRAM, WiFi, active chips)
GET/api/statsRead/write/erase counters for SPI and I2C
POST/api/stats/resetReset all statistics

ROM Slots

GET/api/slotsList all 4 slots with metadata
POST/api/slots/{n}/upload?chip={type}&label={name}Upload ROM image (raw binary body, Content-Type: application/octet-stream)
GET/api/slots/{n}/downloadDownload ROM image as binary
POST/api/slots/{n}/insertActivate slot (starts emulation on its bus)
POST/api/slots/{n}/ejectDeactivate slot (stops emulation)
DELETE/api/slots/{n}Delete ROM image (must eject first)
POST/api/slots/{n}/labelSet label — body: {"label":"name"}

Chip Configuration

GET/api/chipList supported SPI flash and I2C EEPROM chips with JEDEC IDs
POST/api/chipSet active chip — body: {"type": 4} (chip_type_t enum value)

WiFi

GET/api/wifiCurrent WiFi mode, IP, RSSI
POST/api/wifiSet STA credentials — body: {"ssid":"...","password":"..."}

GPIO Control

GET/api/gpioCurrent reset & power pin state
POST/api/gpio/reset/assertHold target in reset
POST/api/gpio/reset/releaseRelease target from reset
POST/api/gpio/reset/pulsePulse reset — body: {"duration": 100} (ms)
POST/api/gpio/reset/sequenceAssert, wait, release, wait — body: {"duration": 100, "post_delay": 100}
POST/api/gpio/power/onPower on (MOSFET gate high)
POST/api/gpio/power/offPower off (MOSFET gate low)
POST/api/gpio/power/cyclePower off, wait, on — body: {"duration": 500, "settle": 200}

Live Events (SSE)

GET/api/eventsServer-Sent Events stream (max 4 clients)
GET/api/logLast 100 access log entries as JSON array

SSE Event Types

access — emitted per bus transaction (batched every 50ms)

{"ts":12345,"addr":"0x000000","len":256,"op":"read","bus":"spi","cmd":"0x0B","slot":0}

stats — emitted every 2s

{"spi_reads":100,"spi_writes":0,"spi_erases":0,"spi_bytes_read":25600,...}

status — emitted every 5s

{"heap_free":200000,"psram_free":7000000,"uptime":3600,"rssi":-45,"ip":"192.168.1.42"}

Example: Upload & Insert via curl

## Upload a ROM image to slot 0 as W25Q128 (type=4)
curl -X POST "http://romemu.local/api/slots/0/upload?chip=4&label=my_firmware" \
     -H "Content-Type: application/octet-stream" \
     --data-binary @firmware.bin

## Insert slot 0 (starts SPI flash emulation)
curl -X POST http://romemu.local/api/slots/0/insert

## Check status
curl http://romemu.local/api/status

## Stream live access events
curl -N http://romemu.local/api/events

## Eject when done
curl -X POST http://romemu.local/api/slots/0/eject
